# 龍馬館売上目標・実績管理ポータル

BAR「龍馬館」の下半期（7月〜12月）における売上目標の進捗と、曜日ごとの分析を行うための売上管理ポータルです。

## 主な機能
1. **ダッシュボード**：下半期全体の目標達成状況、当月の累積売上進捗トレンド、客単価、週次目標の達成度をグラフとインジケーターで可視化します。
2. **月別実績入力シート**：日々の売上・客数・イベントタイプを入力できます。週次での集計行や進捗率も自動計算されます。
3. **曜日・客数分析**：入力データから曜日ごとの平均売上、客数、客単価（¥/人）を動的に集計し、グラフ及び表で分析します。
4. **計算モードの切り替え（設定画面）**：
   - **比率調整モード**：曜日別の目標比率を保ちながら、月間合計が目標金額（例: 7月=70万）にピッタリ一致するよう日別目標を配分します。
   - **絶対値モード**：指定された単価（例: イベント日=6万、水日=1.5万）をそのまま毎日適用し、月次目標は7月に対する売上比率でスケールされます。
5. **ローカル永続化とエクスポート**：入力データはブラウザに自動保存されます。またJSONファイルでのエクスポート・インポートが可能です。

## 動かし方

このプロジェクトは完全に静的なHTML/CSS/JSで構成されているため、特別な環境構築は不要です。

### 1. 直接ブラウザで開く場合
`index.html` ファイルをブラウザ（Chrome, Safari, Edgeなど）にドラッグ＆ドロップするか、ダブルクリックして開くことで、すぐに利用できます。

### 2. ローカルサーバーを使う場合（推奨）
ブラウザのセキュリティ上の制限（ファイル共有等）を回避し、よりスムーズに動作させるため、ローカルサーバー経由での起動が推奨されます。

#### Python を使う場合
ターミナルを開き、このフォルダ内で以下のコマンドを実行します：
```bash
python3 -m http.server 8000
```
起動後、ブラウザで `http://localhost:8000` にアクセスしてください。

#### VS Code プラグインを使う場合
VS Code をお使いの場合は、拡張機能「**Live Server**」をインストールし、画面右下の「Go Live」をクリックすることで起動できます。
